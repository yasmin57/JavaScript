Relatório

API: https://pokeapi.co/

Consumindo a API

1º Passo: "Guarde" a url

Descrição: Crie uma variável ou uma constante do tipo string e atribua a ela o seguinte valor: https://pokeapi.co/api/v2/pokemon/

Demonstração: const $url = "https://pokeapi.co/api/v2/pokemon/";

2º Passo: Utilize o fetch

Descrição: O fetch devera receber como argumento a url da API e o nome do pokemon desejado.

Demonstração: fetch(url + 'pikachu')

3º Passo: Use o método then

Descrição: Transforme a promessa em json e guarde em uma variável

Demonstração: fetch(url + nome_do_Pokemon)
.then(resposta => resposta.json())
.then(dados => {jsonPokemon = dados;})

Sugestão/Código final

Descrição: Coloque isso em uma função que receba a URL e o nome do Pokemon

Demonstração:
// url da api
const $url = "https://pokeapi.co/api/v2/pokemon/";

// variaveis
var name, jsonPokemon;

// requisição da API
const requisicaoAPI = (url, name) =>{
fetch(url + name)
.then(resposta => resposta.json())
.then(dados => {jsonPokemon = dados;})
}
